# Datasets

This directory is for storing dataset files.

## Note
- Large dataset files are excluded from git tracking (see `.gitignore`)
- Download datasets as needed for each lab exercise
- Common datasets: MNIST, CIFAR-10, ImageNet, etc.

## Dataset Sources
- [Kaggle](https://www.kaggle.com/datasets)
- [UCI Machine Learning Repository](https://archive.ics.uci.edu/ml/index.php)
- [PyTorch Datasets](https://pytorch.org/vision/stable/datasets.html)
