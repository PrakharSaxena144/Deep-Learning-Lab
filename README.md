# Deep Learning Lab

This repository contains implementations and experiments from Deep Learning Lab exercises.

## Repository Structure

```
Deep-Learning-Lab/
├── Lab1/          # Neural Network Basics
├── Lab2/          # Convolutional Neural Networks
├── Lab3/          # Recurrent Neural Networks
├── Lab4/          # Advanced Architectures
├── Lab5/          # Transfer Learning
├── Lab6/          # Autoencoders and GANs
├── datasets/      # Dataset files (not tracked by git)
└── utils/         # Common utility functions
```

## Getting Started

### Prerequisites
- Python 3.8 or higher
- PyTorch or TensorFlow
- NumPy, Pandas, Matplotlib

### Installation

```bash
pip install -r requirements.txt
```

## Usage

Navigate to the specific lab directory and run the Python scripts:

```bash
cd Lab1
python experiment.py
```

## Labs Overview

- **Lab1**: Introduction to Neural Networks and Perceptrons
- **Lab2**: Convolutional Neural Networks for Image Classification
- **Lab3**: Recurrent Neural Networks for Sequence Data
- **Lab4**: Advanced Architectures (ResNet, VGG, etc.)
- **Lab5**: Transfer Learning and Fine-tuning
- **Lab6**: Generative Models (Autoencoders, GANs)
